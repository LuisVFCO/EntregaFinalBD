# Entrega Final - Big Data

## Descrição do Projeto
- O projeto é o tratamento e a análise estatística de dados do Dataset do PySus, uma biblioteca em Python desenvolvida para facilitar o acesso e a análise de dados do Sistema Único de Saúde (SUS) do Brasil. Utilizando o Sistema de Informação de Agravos de Notificação (SINAN), o Dataset escolhido pelo grupo foi sobre casos de tuberculose no ano de 2022.
  
- As ferramentas utilizadas para o desenvolvimento do projeto foram o Python e Apache Spark sendo, o projeto, desenvolvido no Google Colab.
